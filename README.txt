USAGE:
./main.py [query file] [directory containing query corpus]

Files may be in any format readable by tuneR's readWave function

EXAMPLE:
./main.py sampleDatabase/tabla2.wav sampleDatabase

OUTPUT FORMAT:
1: tabla2.wav - Distance: 0.0
2: tabla3.wav - Distance: 0.196380031844
3: snare.wav - Distance: 0.229155635903
